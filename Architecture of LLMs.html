Architecture of LLMs

Diving deeper into the deep learning technique, it enables machine learning by utilizing a multi-layered network which is called “Deep neural network” (IBM, n.d.). The LLM employs Deep neural networks with the transformer-based architectures for processing and generating in a form humans can understand (Datagen, n,d.). A neural network typically includes three types of layers (Gavril Ognjanovski, 2019):

  1. Input layer
  2. Hidden layer
  3. Output layer

Within the layers, interconnected nodes are contained which perform computation or transformation upon inputs (DevX, 2023). Hidden layers play a key role in deep learning. The more the hidden layers, the more depth of a network, while the more neurons present in a layer, the more the width, each of the hidden layers enabling the network to learn complex representations gradually (DeepAI, n.d.). To be specific, features, patterns and relationships are learnt through input data, eventually having the capability to make predictions and proper outputs (DevX, 2023).

On the other hand, Natural Language Processing (NLP) plays the role of dealing with interaction between human language and computers, it can be divided into two parts: (1) Natural Language Understanding and (2) Natural Language Generation (DeepLearningAI, 2023).

Last but not least, it is worth noting that even the experts did not fully understanding the inner workings of a LLM (Dhanshree Shripad Shenwai, 2023)
